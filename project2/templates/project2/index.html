Project 2: Text Classification & Active Learning

This project builds a text classifier on IMDB reviews and enhances its performance through active learning, in which the model queries new labels using different strategies.

Passive Text Classification

We train on 70% of the IMDB reviews (in project2/data/imdb.csv(.gz)) and test on the remaining 30%.
Representation: TF-IDF (unigrams & bigrams, top 20,000 features)
Classifier: Logistic Regression
